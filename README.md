# 📊 Lead Tracker & Outreach Automation System

## 🔍 What I Built
A complete outbound sales lead tracking system built 
from scratch in Google Sheets — designed to help 
founders and sales teams manage outreach efficiently.

## ⚙️ Features
- Lead database with 10+ tracked fields
- Color-coded status tracking (New/Contacted/Qualified)
- Auto-calculated KPI Dashboard:
  - Total Leads
  - Outreach Sent
  - Response Rate %
  - Qualified Leads
- Follow-up date tracking
- Source attribution (LinkedIn/Cold Email/Referral)

## 📈 Dashboard Metrics
| Metric | Value |
|--------|-------|
| Total Leads | 10 |
| Outreach Sent | 7 |
| Response Rate | 57.1% |
| Qualified Leads | 3 |

## 🛠️ Tools Used
- Google Sheets
- Conditional Formatting
- Formula Automation (COUNTIF, COUNTA, ROUND)



## 💡 Why I Built This
Outbound sales teams waste hours in manual tracking.
This system automates status visibility and response 
rate calculation — giving founders real-time clarity 
on their pipeline.
